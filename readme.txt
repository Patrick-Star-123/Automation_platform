#注：平台基于airtest开发，需安装airtest、pocoui等网易airtest的相关库只可用于安卓设备
#配置文件：
1、运行前需检查配置文件，各项地址是否正确（地址为所需文件绝对路径）；
2、所有配置信息都为必填项；
3、标点符号需按格式填写；
4、多设备执行时需使用批处理方式且填写‘测试机用例配置’。
#测试用例：
1、用例内容需按格式填写，且格式不可修改
2、用例标题建议简短，否则报告输出时可能展示不完全或展示异常
3、分隔符‘丨’不可修改，且每条用例最后也需加分隔符
4、【当前操作】用途为报告展示，【动作】用途为判断当前所做操作，【按钮地址类型】用途为报告展示，【按钮操作/坐标/截图文件名】用途为填写执行操作的poco代码或坐标、截图名称，【输入的值/随机字符长度】当为输入的值时做展示用途，当为随机字符长度时填写所生成的随机字符长度，【等待时间】用途为填写这一步骤点击前所需等待时间，避免页面加载较慢导致用例异常。
5、当为截图动作时截图文件需存放在项目文件夹【button_screenshot】下
6、动作含：
【点击】普通点击事件，直接复制airtest-ide内相应操作代码即可；
【输入】普通输入事件，复制airtest-ide内相应操作代码，所输入内容需填写在代码内；
【输入随机数】输入的内容为随机数，复制airtest-ide内相应操作代码，代码内输入内容必填，但内容不限，会被随机数替换，输入随机字符长度可控制随机数长度；
【滑动】普通滑动事件，直接复制airtest-ide内相应操作代码即可;
【坐标点击】使用屏幕绝对坐标点击，填写的坐标需开启手机开发者选项-指针位置，内容为X、Y轴数据；
【截图点击】需提前保存点击位置截图，填写截图名称至用例即可；
【滑动截图】使用截图操作滑动，须填写截图名称，以及滑动数据，三个滑动数据可在airtest-ide内查看并按格式填写；
【按键点击】填写按键名称即可；
【断言】判断元素存在，存在即返回True，不存在即返回False；
【假断言】判断元素不存在，存在即返回False，不存在即返回True；
